Who is this for?

This session is designed for students age 10 and up. Must have access to a personal computer.

Why learn to code?

Everyone should learn how to code. To put simply, coding is the language of the future. The earlier that kids learn this language, the better equipped they are for problem solving, career development, creativity, digital literacy, and teamworking skills.

Learning Objective

The Winter session camp will teach Python, a popular programming language used in many different applications. Today, Python is the 2nd most sought after language on the internet, and one of the fastest growing coding languages.

Games

Python is a widely used and preferred language for game development. Snake, Civilization IV, Sims IV, and EVE Online are all examples of mainstream games developed in Python.

Scripts

One of the most versatile features of Python, the language is used to write code to perfom tasks automatically. This feature of the langauge is used most commonly in System Administrator careers.

Security

Learning to use Python along with the best security practices is a must. During the winter coding camp, students will learn industry level security practices.

Web Development

Python is used to upkeep the most visited websites in the world, including names like Google, Netflix, Uber, and Spotify.

Data Structures

This course will cover basic data structures in Python, knowledge that is essential for any aspiring software engineer. Data structures are essential for building powerful applications and parsing large amounts of data.

Algorithms

This course places emphasis on algorithms, as students will build and implement the most essential ones. knowledge of algorithms is essential to building any app that is running at maximum efficiency.
